4 ui UI

4.1 ui.button Button

A button is a clickable element that perfoms an action.

Link
.button--encouraging – Buttons that perform actions which are recommended
Link
.button--disencouraging – Buttons that perform actions which that could be considered harmful
Link
.button--passive – Buttons that perform little or no actions
Link 
<a href="#" class="button [modifier class]">Link</a>
<button type="submit" class="button [modifier class]">Button</button>
<input type="submit" class="button [modifier class]" value="Input:submit" />
<input type="button" class="button [modifier class]" value="Input:button" />

4.2 ui.form-actions Form actions

A wrapper for buttons eg. at the bottom of a form.

Cancel 
<div class="form-actions">
  <span class="form-actions__primary">
    <input type="submit" class="button" value="Save &amp; continue editing">
    <input type="submit" class="button button--encouraging" value="Save">
  </span>
  <span class="form-actions__secondary">
    <a href="#" class="button button--passive">Cancel</a>
  </span>
</div>

4.3 ui.quick-form Quick form

A form that only requires one text field and one button

<form class="quick-form">
  <input type="text" placeholder="Barcode" />
  <input type="submit" class="button" value="add" />
</form>
