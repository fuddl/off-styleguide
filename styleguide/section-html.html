2.1.8 html.block.lists Lists

2.1.8.1 html.block.lists.ol Ordered list

Use a numbered block component on individual list items to mark, which Number or Letter shall be used to mark the item.

  1. a) Veggies es bonus vobis
  2. b) proinde vos postulo essum
  3. c) magis kohlrabi welsh
  4. d)

    onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.

    Gumbo beet greens corn soko endive gumbo gourd. 

<ol>
  <li class="numbered">
    <span class="numbered__number">a)</span>
    Veggies es bonus vobis
  </li>
  <li class="numbered">
    <span class="numbered__number">b)</span>
    proinde vos postulo essum
  </li>
  <li class="numbered">
    <span class="numbered__number">c)</span>
    magis kohlrabi welsh
  </li>
  <li class="numbered">
    <span class="numbered__number">d)</span>
    <p>onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.</p>
    <p>Gumbo beet greens corn soko endive gumbo gourd. </p>
  </li>
</ol>

2.1.8.2 html.block.lists.ul Bulletpoint list

A list without any modifiers will use bullet points (•) to mark list items.

  • Veggies es bonus vobis
  • proinde vos postulo essum
  • magis kohlrabi welsh

  • onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.

    Gumbo beet greens corn soko endive gumbo gourd. 

<ul>
  <li>Veggies es bonus vobis</li>
  <li>proinde vos postulo essum</li>
  <li>magis kohlrabi welsh</li>
  <li>
    <p>onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.</p>
    <p>Gumbo beet greens corn soko endive gumbo gourd. </p>
  </li>
</ul>

2.1.9 html.block.p Paragraph

The HTML <p> element (or HTML Paragraph Element) represents a paragraph of text. Paragraphs are usually represented in visual media as blocks of text that are separated from adjacent blocks by vertical blank space and/or first-line indentation. Paragraphs are block-level elements.

Veggies es bonus vobis, proinde vos postulo essum magis kohlrabi welsh onion daikon amaranth tatsoi tomatillo melon azuki bean garlic. 

<p>
  Veggies es bonus vobis, proinde vos postulo essum magis kohlrabi welsh
  onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.
</p>
